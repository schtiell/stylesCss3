CSS3: Cascading Style Sheets Propiedades de las fuentes

Las propiedades relacionadas con las fuentes son tipo de letras, caracteristicas, tamaños, familias entre otras

Algunas de las propiedades css3 son:

  • font-family: propiedad para especificar la fuente de un texto. Css tiene 5 familias genericas: "serif", "san-serif", "monospace", "cursive" y "fantasy"
  • font-size: propiedad establece el tamaño del texto, puede especificarse el tamaño en "px", "em", "%" o "vw"
  • font-style: propiedad se utiliza principalmente para especificar texto en cursive, italic, oblique o normal
  • font-weight: la propiedad establece el peso (o negrita) de la fuente. Los pesos disponibles dependen del que esté configurado actualmente: "normal", "bold", "lighter", "bolder", "100", "900"
  • font-variant: la propiedad permite establecer todas las variantes de fuente para una fuente. "small-caps", "normal" entre otros

La siguiente lista de titulos utiliza los siguientes propiedades aplicadas a las fuentes:

		<styles>
		h1{
			font-size: 40px;
			font-family: verdana;
			font-variant: small-caps;
		}
		h2{
			font-size: 80%;
			font-weight: bold;
		}	
		h3{
			font-size: 30px;
			font-style: oblique;
			font-weight: lighter;
		}
		h4{
			font-size: 1.5em;
			font-style: italic;
		}
		h5{
			font-size: 20px;
			font-family: tahoma;
			font-weight: 300;
		}
		h6{
			font-size: 15px;
			font-style: oblique;
			font-variant: small-caps;
		}
		</styles>

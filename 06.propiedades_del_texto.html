CSS3: Cascading Style Sheets Propiedades elacionadas a los textos

Para dar formato a los textos existen propiedades, como:

  • color: define el color del texto, recibe valores de colores en "ingles", "rgba" o "hexadecimal"
  • text-align: propiedad que define la alineacion horizontal del texto, puede estar alineado: "left", "right", "center" o "justify"
  • text-decoration: se utiliza para agregar una linea de decoraci√≥n, es decir tipos de subrayado: "overline", "underline", "line-through", "overline underline"
  • text-transform: propiedad utilizada para transformar los textos, mayusculas, minusculas o letra capital, "uppercade", "lowercase" o "capitalize"
  • text-spacing: esta propiedad se utiliza para especificar espacios
    1. text-indent: Especifica una sangria en la primera linea
    2. letter-spacing: Especifica espacio entre letras
    3. line-height: Especifica espacio entre lineas
    4. word-spacing: Especifica espacio entre palabras
    5. white-space: Especifica como se maneja el espacio en blanco de un elemento
  • text-shadow: Agrega una sombra al texto

Los estilos aplicados en los elementos HTML son los siguientes: 

		h1{
			color: #A569BD ;
			text-align: right;
			text-decoration: overline;
			text-transform: capitalize;
			text-align: center;
		}
		
		h2{
			color: #E74C3C ;
			text-align: left;
			text-decoration: underline;
			text-shadow: 2px 3px #EB984E;
		}
		
		p{
			text-decoration: line-through;
			letter-spacing: 5px;
			word-spacing: 20px;
			text-align: justify;
		}

CSS3: Cascading Style Sheets Herencia de propiedades de estilos

La herencia en CSS es el mecanismo mediante el cual determinadas propiedades de un elemento padre se transmiten a sus hijos. De hecho, se parece mucho a la herencia genética. Si los progenitores tienen los ojos azules, los hijos seguramente también tendrán los ojos azules.

La herencia de las propiedades de los estilos se aplican a todos los elementos o etiquetas dentro de otras etiquetas, es decir la herencia debe entenderse en este contexto: algunos valores de las propiedades css que se han establecido para los elementos padre los heredan los elementos hijo, pero otros no

Por ejemplo los márgenes no son heredables porque es poco probable que un elemento hijo necesite los mismos márgenes que su padre. Normalmente, el sentido común dicta qué propiedades se heredan y cuáles no

Todos los elementos de un documento HTML heredan todas las propiedades heredables de su padre excepto el elemento raíz (html), que no tiene progenitor.

El hecho de que las propiedades heredadas tengan algún efecto o no depende de otros factores, como veremos más adelante cuando hablemos de la cascada. De la misma manera que una madre de ojos azules puede tener un hijo de ojos marrones si el padre tiene los ojos marrones, las propiedades heredadas en CSS pueden anularse.

La herencia puede forzarse utilizando la palabra clave inherit (heredar) incluso para propiedades que no son heredables normalmente.

Con las propiedades abreviadas se puede utilizar inherit en vez de los valores normales.

Se debe utilizar la versión abreviada o bien para todo o bien para nada en absoluto. En la versión no abreviada no se pueden especificar algunos valores y utilizar inherit para otros porque los valores pueden darse en cualquier orden y no hay manera de especificar qué valores queremos heredar.

Los siguientes parrafos heredan los estilos definidos para el contenedor padre, hasta que se define un estilo diferente para el tercer parrafo, de acuerdo a lo visto con el termino: especificidad

		.ejercicio{
			color: #E74C3C ;
			font-family: tahoma;
			text-indent: 30px;
		}

		.parrafo{
			color: #000;
			font-family: verdana;
			text-indent: 0%;
			text-transform: uppercase;
		}

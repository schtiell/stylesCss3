CSS3: Cascading Style Sheets Herencia de propiedades de estilos

La herencia de las propiedades de los estilos se aplican a todos los elementos o etiquetas dentro de otras etiquetas, es decir la herencia debe entenderse en este contexto: algunos valores de las propiedades css que se han establecido para los elementos padre los heredan los elementos hijo, pero otros no

Los siguientes parrafos heredan los estilos definidos para el contenedor padre, hasta que se define un estilo diferente para el tercer parrafo, de acuerdo a lo visto con el termino: especificidad

		.ejercicio{
			color: #E74C3C ;
			font-family: tahoma;
			text-indent: 30px;
		}

		.parrafo{
			color: #000;
			font-family: verdana;
			text-indent: 0%;
			text-transform: uppercase;
		}

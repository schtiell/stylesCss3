CSS3: Cascading Style Sheets Propiedades de bordes

Es importante saber que todo elemento que se crea dentro de una página HTML genera una caja. Por ejemplo si se han creado elemento h1, p, li, etc, y se les fija la propiedad background-color podrá verse que el contenido se encuentra dentro de un rectángulo.

A este rectangulo que cubre cada elemento html se le conoce como box model o modelo de caja

De acuerdo con este modelo de caja se puede aprender sobre los limites o fronteras que tiene cada elemento, a estos se les conoce como bordes y son estos los que afectan o definen el tamaño de cada elemento

Las propiedades de borde son utilizadas para definir el ancho, el estilo y el color, son las siguientes

  • border-width: Define el ancho del borde, puede definirse en px, pt, cm, em, etc, propiedad puede tener de uno a cuatro valores: borde (superior, derecho, inferior e izquierdo)
  • border-style: Propiedad especifica qué tipo de borde mostrar, dotted, dashed, solid, double, entre otros.La propiedad puede tener de uno a cuatro valores: borde (superior, derecho, inferior e izquierdo)
  • border-color: Propiedad se utiliza para establecer el color de los cuatro bordes, el color puede ser definido: nombre, HEX, rgb, rgba, hsl. La propiedad puede tener de uno a cuatro valores: borde (superior, derecho, inferior e izquierdo)

y son utilizadas de la siguiente manera:

		.box{
			border-width: 3px 16px 8px 21px;
			border-style: dotted dashed solid double;
			border-color: #52BE80 #F7DC6F #AF7AC5 #E74C3C;
		}

		.box2{
			border-width: 3px 16px;
			border-style: dotted dashed;
			border-color: #52BE80 #F7DC6F;
		}

		.box3{
			border-width: 3px;
			border-style: solid;
			border-color: #52BE80;
		}

Importante: Si las propiedades solo especifican 2 de los 4 posibles valores, se le está indicando al navegador que el primer valor son para (arriba y abajo), si solo es 1 valor entonces se aplica el mismo para los 4 lados

Es posible tambien especificar el ancho, el estilo y el color de cada borde en propiedades separadas, por ejemplo:

		border-top-width: 1px;
		border-left-width: 1px;
		border-bottom-width: 1px;
		border-right-width: 1px;

		border-top-style: solid;
		border-left-style: dashed;
		border-bottom-style: double;
		border-right-style: dotted;

		border-top-color: #212F3D;
		border-left-color: #52BE80;
		border-bottom-color: #AF7AC5;
		border-right-color: #E74C3C;

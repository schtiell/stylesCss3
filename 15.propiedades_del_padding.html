CSS3: Cascading Style Sheets Propiedades del padding

El padding o relleno es utilizado para crear un espacio alrededor del contenido de un elemento, dentro de los bordes definidos por el box model.Es decir se genera yb espacio entre el contenido del elemento HTML y la frontera o borde

Con CSS se tiene un control total sobre el padding. Existen propiedades para establecer el relleno para cada lado del elemento (top, left, bottom y right)

  • padding-top: Relleno superior del elemento
  • padding-right: Relleno izquierdo del elemento
  • padding-bottom: Relleno inferior del elemento
  • padding-left: Relleno derecho del elemento

Se pueden especificar los 4 valores, 2 valores o bien 1 valor a la propiedad padding.

		div {
			padding-top: 50px;
			padding-right: 30px;
			padding-bottom: 50px;
			padding-left: 80px;
		}

Si se especifican solo 2, el primer valor se aplica al padding superior e inferior y el segundo valor al paddign derecho e izquierdo.

		div {
			padding: 50px 30px;
		}

Si solo un valor es indicado, significa que ese padding debe aplicarse a los 4 lados de la caja

		div {
			padding: 30px;
		}

Todas la propiedades de padding pueden tener los siguientes valores:

  • longitud: Especifica un padding en px, pt, em, cm, etc
  • %: Especifica un padding en % del ancho del elemento contenedor
  • inherit: Especifica que el padding se debe heredar del elemento principal

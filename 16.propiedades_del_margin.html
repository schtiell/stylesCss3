CSS3: Cascading Style Sheets Propiedades del margin

Los márgenes se utilizan para crear espacio alrededor de los elementos, fuera de los bordes definidos.

Con CSS, tienes control total sobre los márgenes utilizando la propiedad margin. Hay propiedades para establecer el margen de cada lado de un elemento (superior, derecho, inferior e izquierdo).

  • margin-top: Margen superior
  • margin-right: Margen derecho
  • margin-bottom: Margen inferior
  • margin-left: Margen izquierdo

A igual que el padding las propiedades de margen pueden especificarse una por una, son:

		.contenedor{
			margin-top: 20px;
			margin-right: 30px;
			margin-bottom: 15px;
			margin-left: 25px;
		}

Si solo 2 propiedades son especificadas entonces se aplica el primero para el margen superior e inferior y el segundo valor para el margen izquierdo y derecho

		.contenedor{
			margin: 20px 30px;
		}

Si solo es especificado un valor, este será aplicado a los 4 lados:

		.contenedor{
			margin: 20px;
		}

El margen está después del borde. El margen separa un elemento HTML de otro elemento HTML dentro de la página.

  • longitud: Especifica un padding en px, pt, em, cm, etc
  • %: Especifica un padding en % del ancho del elemento contenedor
  • inherit: Especifica que el padding se debe heredar del elemento principal

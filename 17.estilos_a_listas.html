CSS3: Cascading Style Sheets Propiedades de listas

En html existen 2 tipos de listas que son usadas para enumerar una serie de elementos. Se utiliza el elemento HTML <ul> </ul> para generar listas no ordenadas. Listas en viñetas. Se utiliza el elemento HTML <ol> </ol> para generar listas ordenadas. Listas numeradas

Y cada item de la lista con el elemento HTML <li> </li>

Las propiedades de la lista CSS permiten:

  • Establecer diferentes marcadores de elementos de lista para listas ordenadas
  • Establezca diferentes marcadores de elementos de lista para listas desordenadas
  • Establecer una imagen como marcador de elemento de lista
  • Agregar colores de fondo a listas y elementos de lista

Css tiene su manera de tratar con las propiedades de ambas listas:

  • list-style-type: propiedad especifica el tipo de marcador de elemento de lista.
  • list-style-position: especifica si los marcadores de elementos de lista deben aparecer dentro o fuera del flujo de contenido.
  • list-style-image: propiedad especifica una imagen como marcador de elemento de lista.

Los valores para list-style-type son:

  • none
  • disc
  • circle
  • square
  • decimal
  • decimal-leading-zero
  • lower-roman
  • upper-roman
  • lower-alpha
  • upper-alpha

Los valores para list-style-position son:

  • inside
  • outside

Los valores para list-style-image son:

  • none
  • url

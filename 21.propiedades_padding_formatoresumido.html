CSS3: Cascading Style Sheets Propiedades del padding metodo corto

Podemos inicializar el padding de un elemento HTML en forma resumida con:

		selector{
			padding: 5px 10px 12px 3px;
		}
		selector{
			padding: padding-top padding-right padding-bottom padding-left;
		}

Si se indican dos valores unicamente, el primero se aplica a padding-top padding-bottom y el segundo valor a padding-right y padding-left

		selector{
			padding: 5px 10px;
		}

Si se indica un Ãºnico valor se aplica a los cuatro lados:

		selector{
			padding: 1px;
		}

Ejemplo

	.ejemplo1{
		padding: 10px 20px 30px 40px;
	}
	
	.ejemplo2{
		padding: 10px 20px;
	}
	
	.ejemplo3{
		padding: 30px;
	}

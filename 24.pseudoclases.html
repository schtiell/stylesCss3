CSS: Cascading Style Sheets Pseudoclases

Las pseudoclases son unas clases especiales, que se refieren a algunos estados del elemento HTML, las que se utilizan fundamentalmente son las que se aplican al elemento <a>, aunque pueden aplicarse a demas elementos.

			a:pseudoclase {
				propiedad: valor;
			}

Es decir separamos el nombre del selector css con dos puntos seguido del nombre de la pseudoclase. Para el elemento HTML <a> se tienen 4 pseudoclases fundamentales:

  • link: enlace sin visitar
  • visited: enlace presionado
  • hover: enlace que tiene el apuntador del mouse encima
  • active: tiene el foco en ese momento

Pseudoclases CSS

Las pseudoclases son unas clases especiales, que se refieren a algunos estados del elemento HTML, las que se utilizan fundamentalmente son las que se aplican al elemento <a> (enlaces).

Es decir separamos el nombre del elemento HTML con dos puntos. Para el elemento HTML <a> tenemos 4 pseudoclases fundamentales:

  • link: enlace sin visitar
  • visited: enlace presionado
  • hover: enlace que tiene el apuntador del mouse encima
  • active: es la que tiene el foco en ese momento

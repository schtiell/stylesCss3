Unidades de Medida CSS

Css dispone de varios tipos de medidas

  • px: pixeles
  • rem: tamaño de la fuente respecto al elemento raíz del documento
  • em: altura de la fuente por defecto
  • ex: altura de la letra por minuscula
  • in: pulgadas
  • cm: centimetros
  • mm: milimetros
  • pt: puntos, 1 punto es igual que 1/72 puladas
  • %: porcentaje

Las unidades con longitudes relativas son:

  • rem
  • em
  • %

Estas unidades relativas escalan mejor en celulares, tablets y monitores

El framework de CSS Bootstrap 4 utiliza en forma masiva la unidad de medida "rem", si abrimos su código fuente veremos que se inicializan más de 600 propiedades utilizando esta medida.

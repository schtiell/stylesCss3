CSS: Cascading Style Sheets Formas para indicar el color

Para asignar el color hexadecimal se indican los valores correspondientes al rojo,verde y azul precedido por el caracter #

			selector{
				background-color:#ff0000;
			}

Otras sintaxis para indicar el color:

			selector{
				background-color:rgb(255,0,0);
			}

Utilizando la función rgb pasando como parámetro la cantidad de rojo, verde y azul en formato decimal (un valor entre 0 y 255).

También con la función rgb podemos indicar un porcentaje entre 0% y 100% para cada componente:

			selector{
				background-color:rgb(100%,0%,0%)
			}

Por último en algunas situaciones se puede utilizar una sintaxis reducida para ciertos valores:

			selector{
				background-color:#ffaaff;
			}

			selector{
				background-color:#faf;
			}

Es decir si cada valor hexadecimal está formado por el mismo caracter o número, se puede utilizar esta sintaxis simplificada indicando un caracter o número solamente.

Para el siguiente ejemplo no se puede usar ya que los últimos 2 caracteres, fa, son distintos.

			selector{
				background-color:#ffaafa
			}

Ejemplo:

				.color1{
					background-color: #faa;
					color: rgb(44,14,63);
					border: 1px solid hsl(217, 77%, 30%);
				}
				
				.color2{
					color: white;
					background: rgb(44,14,63);
				}
				
				.color3{
					background-color: rgba(76,155,244,.6);
					color: fff;
					border: 1px solid #0B49AD;
				}

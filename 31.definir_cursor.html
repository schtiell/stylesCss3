CSS: Cascading Style Sheets Definir tipos de cursor

Disponemos de una propiedad llamada cursor que tiene por objetivo definir el cursor a mostrar cuando la flecha del mouse se encuentra sobre el elemento HTML. Los valores que podemos asignarle a esta propiedad cursor son:

  • crosshair: cursor cruzado, a menudo utilizado para indicar la selección en un mapa de bits
  • default: Cursor predeterminado (típicamente una flecha)
  • pointer: Cursor usado sobre enlaces (típicamente una mano)
  • move: El objeto suspendido puede moverse
  • e-resize: Cursor para cambiar el tamaño del borde derecho de un cuadro
  • ne-resize: Cursor para cambiar el tamaño de la esquina superior derecha de un cuadro
  • nw-resize: Cursor para cambiar el tamaño de la esquina superior izquierda de un cuadro
  • n-resize: Cursor para cambiar el tamaño del borde superior de un cuadro
  • se-resize: Cursor para cambiar el tamaño de la esquina inferior derecha de un cuadro
  • sw-resize: Cursor para cambiar el tamaño de la esquina inferior izquierda de un cuadro
  • s-resize:Cursor para cambiar el tamaño del borde inferior de un cuadro
  • w-resize: Cursor para cambiar el tamaño del borde izquierdo de un cuadro
  • text: Se puede seleccionar el texto que indica el cursor (típicamente un I-beam)
  • wait:Cursor que indica que el programa está ocupado (a veces un reloj de arena o un reloj)
  • help: Cursor que indica que hay ayuda disponible
  • auto: El browser (user agent) determina el cursor que mostrara basado en el contenido presente.

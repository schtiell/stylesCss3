CSS: Cascading Style Sheets Estilos css para tablas

Otro elemento muy común en un sitio web son las tablas. A este elemento HTML podemos aplicarle varias de las propiedades vistas tanto a la tabla en sí, como sus filas y celdas. El elemento table podemos definir la propiedad border-collapse que puede tomar dos valores:

			selector{
				border-collapse: collapse;
			}

			selector{
				border-collapse: separate;
			}

Si le asignamos el valor collapse une las líneas de los border de las celdas (con esto queda en muchas situaciones mejor estéticamente)

Los siguientes estilos css serán aplicados a la tabla que se presenta al final del sitio

					caption{
						text-align: center;
						padding: 10px;
						border: 1px solid #F4F6F6;
					}
				
					table{
						font: bold 15px Arial;
						border-collapse: collapse;
						border: 1px solid #F4F6F6 ;
						width: 100%;
					}
				
					th{
						background: #AED6F1;
						border-right: 1px solid #FEF9E7;
						border-bottom: 1px solid #FEF9E7;
						width: 25%;
					}

					thead th{
						padding: 15px;
					}
				
					td{
						border: 1px solid #F4F6F6;
						padding: 1em;
						background: #FEF9E7;
						font-weight: lighter;
						text-align: center;
					}
				
					tfoot td{
						font: bold 13px Arial;
						text-align: center;
						background: #E0FFFF;
					}

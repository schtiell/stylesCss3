CSS: Cascading Style Sheets Position: fixed

Esta es otra forma de disponer un elemento dentro de la página. La posición fixed ajusta el elemento en una coordenada fija con respecto a la ventana del navegador. Es una variante del posicionamiento absoluto.

Se debe recordar que con el posicionamiento absoluto se dispone un elemento en una coordenada exacta con respecto a la página, pero con el posicionamiento fixed la posición es con respecto a la ventana del navegador.

En pocas palabras con el posicionamiento fixed si se hace scroll de la página el elemento con dicho valor permanece siempre en el mismo lugar.

Ejemplo:

Este ejemplo despliega 2 barras, una lateral y una barra de navegación superior, ambas con la propiedad position: fixed


				.barraNav{
					position: fixed;
					left: 0px;
				
					width: 100%;
					height: 50px;
				
					background: #000;
					color: #C3C1BF;
				}
				
				.barraLat{
					position: fixed;
					top: 50px;
					left: 0;
				
					width: 200px;
					height: 100%;
				
					background: #F9C77D;
				}

Este ejemplo muestra como aunque se haga scroll al sitio ambas barras con propiedad position: fixed se mantienen en su misma ubicación utilizando todo el navegador.

A diferencia de la position: absolute, la position: fixed, no respeta el tener un contenedor padre, por lo que toma como referencia el navegador. Mientras que la position: absolute toma los valores asignados de top, left, right y bottom a partir del contenedor padre.

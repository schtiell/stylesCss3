Ejemplo:

Este maquetado muestra 2 secciones, la primera un aside la cual declara una clase llamada barra, con los siguientes estilos css:

				position: absolute;
				top: 0;
				left: 0;
				width: 200px;
				height: 100%;
				margin-top:10px;
				background: #E2DED9;

Es es un claro ejemplo de la diferencia con la position: fixed, ya que se puede notar que la seccion aside alcanza su tamaño declarado en css y como su definición es position: absolute y aunque se le indica con height: 100% no hace el scroll al continuar hacia abajo del sitio.

La segunda sección declara las propiedades css siguientes:

				.contenido{
					margin-top: 10px;
					margin-left: 220px;
					margin-right: 20px;
					height: 100%;
				}

Donde se puede notar que unicamente se le indican margenes top, left y right para posicionar el elemento, el cual al no indicarle una propiedad position, su valor es static

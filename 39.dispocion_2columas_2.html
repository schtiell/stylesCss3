CSS: Cascading Style Sheets Disposición de 2 columnas position: absolute

La finalidad de este ejercicio es crear una barra sidebar, la cual muestra un menu con 4 links a diferentes sitios, aplicando la pseudoclase hover a los elementos del menú para aplicar diferentes estilos al pisicionar el puntero del mouse sobre ellos.

Nuevamente el tamaño de alto del sidebar no ocupa todo el navegador ya que su position está declarada como absolute y no fixed, que permitiría utilizar el scroll y la sidebar se mantendría fija durante todo el recorrido del scroll en el sitio

Ejemplo

A la barra sidebar se aplicaron los siguientes estilos

				sidebar{
					position: absolute;
					top: 0;
					left: 0;
					width: 200px;
					height: 100%;
					margin-top:10px;
				}

Para la sección del contenido del ejemplo los estilos declarados unicamente corresponden a margenes y padding

				.contenido{
					margin-top:10px;
					margin-left: 220px;
					margin-right: 20px;
					padding-top: 10px;
				}

Para la lista no ordenada que funcionará para mostrar las opciones del menú: 

				ul{
					display: block;
					width: 80%;
					margin: 20px;
					padding: 0;
				}

Para los elementos de la lista no ordenada, es decir para los list items: 

				ul li{
					display: block;
					margin: 0;
					padding: 0;
					width: 100%;				
					list-style: none;
					border-bottom: 1px solid #fff;	
				}

Para las anclar o hipervinculos, los siguientes estilos:

				ul a{
					display: block;
					margin: 0 auto;
					padding: 10px;
					text-decoration: none;
					background-color: #000;
					color: #D1D1DE;
				}

Por ultimo se aplica una pseudoclase ::hover sobre las anclas del menú para dar la sencación de interaccion al usuario con el menú

				ul a:hover{
					width: 100%;
					background: #6C6EF6 ;
					color: #fff;
				}

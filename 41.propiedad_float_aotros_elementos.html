CSS: Cascading Style Sheets Propiedad float

¿Qué significa flotar?

Para comprender el propósito y el origen de float, podemos mirar el diseño de impresión. En un diseño de impresión, las imágenes se pueden configurar en la página de modo que el texto las envuelva según sea necesario. Esto se denomina comúnmente y apropiadamente "ajuste de texto". Aquí hay un ejemplo de eso.

En los programas de diseño de página, se puede indicar a los cuadros que contienen el texto que respeten el ajuste de texto o que lo ignoren. Ignorar el ajuste de texto permitirá que las palabras fluyan directamente sobre la imagen como si ni siquiera estuviera allí. Esta es la diferencia entre que esa imagen sea parte del flujo de la página (o no). El diseño web es muy similar.

En el diseño web, los elementos de página con la propiedad CSS float aplicada son como las imágenes en el diseño de impresión, donde el texto fluye a su alrededor. Los elementos flotantes siguen siendo parte del flujo de la página web. Esto es claramente diferente a los elementos de página que usan posicionamiento absolute.

Ejemplo

La propiedad float permite como hemos visto con las imágenes tomar un elemento HTML y colocarlo lo más a la izquierda o derecha. Luego los otros elementos siguen el flujo alrededor del elemento que está flotante. Podemos flotar cualquier elemento HTML

Normalmente cuando se utiliza la propiedad float, también debe inicializar la propiedad width para fijar el ancho del elemento.

Cuando se define un elemento que flote el alto del mismo afecta como se disponen los demás elementos de la página, será distinto si es un texto que envolverá al elemento flotante o si se trata de una imagen o div.

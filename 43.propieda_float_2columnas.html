Una breve hisoria de la propiedad float

La propiedad float se introdujo en CSS para permitir a los desarrolladores web crear diseños de estilo de revista, como colocar una imagen dentro de una columna de texto con el texto envolviéndola, similar al diseño de revistas, historietas, periodicos, etc.

La propiedad float se introdujo cuando CSS era un bebé. Es difícil saber cuándo exactamente, ¡pero seguro que float estuvo en los navegadores en 2001! ¡En ese momento, CSS era muy limitado! Cuando las personas querían hacer diseños de página completa, a menudo buscaban float. ¡Era el salvaje oeste en aquel entonces!

De alguna manera, usar float para diseños funcionó. Sin embargo, las lágrimas solían seguir porque esto significaba usar float más allá de su propósito previsto, lo que generaba problemas complicados de código y diseño que eran difíciles de manejar a medida que el sitio web evolucionaba.

¡Hoy en día, simplemente no lo use float para diseños de página! Debe considerar el uso de flotadores de esta manera como una técnica heredada. Flexbox y CSS Grid son opciones muy superiores para crear diseños de página complejos .

Más sobre los valores lógicos

Los valores lógicos nos permiten crear el diseño correcto para idiomas con diferentes direcciones de escritura con un solo valor.

Los valores lógicos más utilizados son lefty right. Los ejemplos que siguen presentan valores lógicos que se usan con menos frecuencia para ilustrar cómo se pueden usar en diferentes casos de uso.

				.header{
					width: 100%;
				}
			
				.sidebar{
					float: left;
					width: 250px;
				}
			
				.article{
					margin-left: 320px;
				}
			
				.footer{
					clear: left;
				}

CSS: Cascading Style Sheets Propiedad float y clear: header, 2 columnas y footer

Una estructura muy común en la web es la disposición de una cabecera de página, seguida de dos columnas y un pie de página.

Se implementará un ancho fluido, es decir que el tamamo del contenido se adapta al tamaño de la ventana del navegador. Esto se consigue iniciando la propiedad width: 100%

La implementación de esta estructura de página es la siguiente:

Ejemplo Explicación de la distribución actual

La primera sección de este sitio corresponde a un encabezado o header. El cual unicamente se le indica un ancho del 100%, para que ocupe por completo el espacio interno de su contenedor padre. Para ello, el padding del padre debe ser 0

La barra lateral utiliza la propiedad float para indicarle que debe flotar con un ancho de 250px

La seccion del articulo inicia con un margen mayor al ancho del siderbar

Por ultimo la seccion de footer inicializa la propiedad clear: left para iniciar en una nueva linea y no a continuación de los nuevos elemetos

Columna: 1 Estilos css aplicados

A continuación se muestran los estilos aplicados para este diseño de header, footer y 3 columnas, aunque solo se muestran los estilos necesarios para la distribución, los estilos referentes a colores, bordes, redondedos etc son omitidos

				.header{
					width: 100%;

				}

				.columna1{
					float: left;
				}
				
				.columna3{
					float: right;
				}
				
				.columna2{
					margin: 0 auto;				
					width: 500px;
				}
				
				.footer{
					clear: left;
					width: 100%;
					
				}

Columna: 3 Explicación estilos css

El primer contenedor en sitio web es el header, al cual se le aplica una clase css header. Donde unicamente se le dice que toma el ancho del 100% con la propiedad width: 100%

El siguiente contenedor es la columna 1, con el mismo nombre de clase css, para el cual se define la propiedad float: left para que flote a la izquierda

Para el contenedor identificado como columna3, se aplica la misma propiedad float pero alineado a la derecha

Para la columna 2, que solo se le indica un margen y un ancho, tomando el cuanta el ancho predefinido para las columnas 1 y 3

Por último el footer, inicia la propiedad clear: left, con la finalidad de iniciar en una nueva linea del DOM, es decir ya no debe de flotar

Columna: 2 Distribución: Header, 3 columnas y footer

Una modificación al concepto anterior nos permite agregar una tercera columna flotando a derecha, lo único que hay que tener cuidado es que dentro del HTML debemos disponer los div de la columna 1 y 3 en primer lugar, ya que son los que se flotan, y por último, la columna 2, que es la central:

Como se indica en el párrafo anterior los elementos HTML que van a utilizar la propiedad float, deben estar antes que los elementos que van a seguir el flujo normal del sitio.

CSS: Cascading Style Sheets Adjacent & General Sibling Selectors

Un selector CSS puede contener mas de un selector simple. Entre los selectores simples, pueden ser incluidos combinators o combinadores

Losadjacent sibling selectors o selectores adyacentes permiten seleccionar un elemento HTML que viene inmediatamente después de otro elemento HTML. Los dos elementos deben tener el mismo elemento padre.

Los selectores adyacentes utilizan el simbolo ( + )

			<body>
				<h1>Titulo <h1>
				<p>Esto es el primer párrafo.</p>
				<p>Esto es el segundo párrafo.</p>
				<p>Esto es el tercer párrafo.</p>
			</body>

Mediante la regla de estilo:

			h1 + p {
				color:#f00;
			}

Seleccionamos el primer párrafo que le sigue al elemento h1 mediante el combinador " + ". Luego el primer párrafo muestra el texto en color rojo

Si aplicamos este otro estilo al mismo bloque HTML:

			p + p {
				color:#f00;
			}

Se pintan de rojo el segundo y tercer párrafo ya que a cada uno de estos párrafos le antecede otro párrafo (no se pinta el primer párrafo porque no le antecede un párrafo sino un "h1"

Ahora los"General Sibling Selectors o Selectores Hermanos Generales" permites seleccionar todos los hermanos a diferencia del selector de hermano adyacente que selecciona el primero. Utiliza el simbolo ( ~ )

Si tenemos el siguiente fragmento:

		<body>
			<h1>Titulo <h1>
			<p>Esto es el primer párrafo.</p>
			<p>Esto es el segundo párrafo.</p>
			<p>Esto es el tercer párrafo.</p>
		</body>

Y le aplicamos la regla:

			h1 ~ p {
				color:#f00;
			}

Como resultado se aplica el color rojo a todos los párrafos adyacentes que tienen como hermano el elemento "h1"

CSS: Cascading Style Sheets Attribute selectors

Otra forma de seleccionar elementos HTML es a traves de sus atributos.

Se debe recordar que un elemento HTML está normalmente constituido por una marca de comienzo, un valor,sus atributos y la marca de cierre

			<a href="http://www.google.com.ar" target="_blank">Titulo Principal</a>

En este ejemplo el elemento "a" tiene definido dos atributos: href y target.

Dada la etiqueta de ancla anterior existe una serie de posibilidades para aplicar selectores css a esta:

Ejemplo

  1. Con el selector de atributo podemos verificar si un elemento HTML tiene definida un atributo determinado para así poder aplicar un estilo. Por ejemplo apliquemos estilos distintos a los elementos "a" que tienen definido el atributo target:

Selector de atributo específico

  1. Google
  2. Facebook
  3. Youtube

Definición del selector

							<ol>
								<li><a href="www.google.com">Google</a></li>
								<li><a href="www.facebook.com">Facebook</a></li>
								<li><a href="www.youtube.com" target="blank">Youtube</a></li>
							</ol>

							a[target]{
								color: red;
								text-decoration: none;
							}

Esta regla se aplica a todos los elementos de tipo "a" que tienen definida la propiedad llamada "target". Es decir el tercer enlace de la página aparece de color rojo ya que tiene definida dicha propiedad. Los dos primeros enlaces no se le aplica esta regla de CSS porque no tienen definida el atributo target.

  1. Una segunda posibilidad es no solo verificar si un elemento HTML define una determinada propiedad sino controlar si su contenido almacena un determinado valor.

Confeccionemos una página que muestre todos los enlaces "a" de la página que tengan inicializado el atributo href con el valor # con una línea tachada (que represente que no tenemos acceso a dicho hipervínculo):

Selector de [atributo="valor"]

  1. Google
  2. Facebook
  3. Youtube

Definición del selector

							<ol>
								<li><a href="#">Google</a></li>
								<li><a href="www.facebook.com">Facebook</a></li>
								<li><a href="#">Youtube</a></li>
							</ol>

							a[href='#']{
								font: oblique lighter 18px tahoma;
								text-decoration: underline;
								color: #00f;
							}

Como se puede ver entre los corchetes no solo hacemos referencia al nombre de la propiedad sino que se dispone el caracter = y la cadena con la que estamos comparando el contenido del atributo.

En este ejemplo se tienen dos enlaces que tienen almacenados en la propiedad href la cadena "#" por lo que dichos enlaces al cumplir la regla del selector se aplica a text-decoration el valor line-trough.

  1. También se puede controlar si el valor de la propiedad comienza, finaliza o tiene en su interior una determinada cadena de caracteres. La sintaxis para este tipo de selectores es:

Selector de [atributo^="valor"]

  1. AppleMusic
  2. Karmany
  3. live mail

Definición del selector

							<ol>
								<li><a href="https://beta.music.apple.com/">Apple music</a></li>
								<li><a href="http://www.karmany.com">karmany</a></li>
								<li><a href="http://www.xbox.com"">xbox</a></li>
							</ol>

							a[href^="https"] {
								color:#f00;
							}

Hay que tener en cuenta que utilizamos la sintaxis ^= para controlar si el atributo comienza con la cadena que le indicamos entre comillas, es decir el primer enlace cumple la regla ya que la propiedad href comienza con esos cinco caracteres https

La sintaxis $= sirve para controlar si el valor del atributo finaliza con la cadena de caracteres indicada.

Selector de [atributo$="valor"]

  1. Apple
  2. Mercadito
  3. live mail

Definición del selector

							<ol>
								<li><a href="http://apple.com/">Apple</a></li>
								<li><a href="http://www.mercadito.com.mx">Mercadito</a></li>
								<li><a href="http://www.xbox.com"">xbox</a></li>
							</ol>
							
							a[href$=".mx"] {
								color:#0f0;
							}

Por último con la sintaxis *= controlamos si el atributo contiene en alguna parte la cadena indicada (en nuestro ejemplo el tercer enlace contiene en su interior la cadena "yahoo"):

Selector de [atributo*="valor"]

  1. Apple
  2. Mercadito
  3. Yahoo

Definición del selector

							<ol>
								<li><a href="http://apple.com/">Apple</a></li>
								<li><a href="http://www.mercadito.com">Mercadito</a></li>
								<li><a href="http://www.yahoo.com"">yahoo</a></li>
							</ol>
							
							a[href*="yahoo"] {
								color: #F06D0C;
							}

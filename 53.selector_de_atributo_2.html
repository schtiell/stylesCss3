CSS: Cascading Style Sheets Attribute selectors

El selector de atributos de CSS coincide con los elementos en funci칩n de la presencia o el valor de un atributo dado. Algunos de los selectores por atributos son:

Su sintaxis:

Representa elementos con un nombre de atributo de attr

			selector[attr]{
				....
			}

Representa elementos con un nombre de atributo de attr cuyo valor es exactamente value .

			selector[attr="vulue"]{
				.....
			}

Representa elementos con un nombre de atributo de attr cuyo valor es una lista de palabras separadas por espacios en blanco, una de las cuales es exactamente value .

			selector[attr~="vulue"]{
				.....
			}

Representa elementos con un nombre de atributo de attr cuyo valor puede ser exactamente el valor o puede comenzar con el valor seguido inmediatamente por un gui칩n -(U+002D). A menudo se usa para coincidencias de subc칩digos de idiomas.

			selector[attr|="vulue"]{
				.....
			}

Representa elementos con un nombre de atributo de attr cuyo valor est치 precedido (precedido) por value .

			selector[attr^="vulue"]{
				.....
			}

Representa elementos con un nombre de atributo de attr cuyo valor tiene el sufijo (seguido) de value .

			selector[attr$="vulue"]{
				.....
			}

Representa elementos con un nombre de atributo de attr cuyo valor contiene al menos una ocurrencia de valor dentro de la cadena.

			selector[attr*="vulue"]{
				.....
			}

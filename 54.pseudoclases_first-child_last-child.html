CSS: Cascading Style Sheets Pseudoclases: first-child y last-child

Otras formas de seleccionar elementos HTML de una página para aplicarle estilos es mediante el uso de las pseudoclases aplicadas a los selectores.

Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado.

Las pseudoclases, junto con los pseudoelementos, permiten aplicar un estilo a un elemento no sólo en relación con el contenido del árbol de documento DOM, sino también en relación a factores externos como el historial del navegador (:visited, por ejemplo), el estado de su contenido (como :checked en algunos elementos de formulario), o la posición del ratón (como :hover que permite saber si el ratón está encima de un elemento o no).

Su sintaxis:

			selector : pseudoclase { 
				propiedad: valor; 
			}

Pseudoclases: first-child & last-child

La pseudoclase fist-child nos permite seleccionar el primer elemento hijo de un elemento padre y de manera similar la pseudoclase last-child selecciona el último elemento hijo.

A las pseudoclases les antecede el caracter : y en forma seguida indicamos el nombre de la pseudoclase, previo a los dos puntos indicamos el nombre del elemento a seleccionar:

			ol li : first-child {
				color: green;
			}

Se puede también expresarlo sin anteceder el selector del elemento <ol>, es decir sin indicar "selector descendiente":

			li : first-child {
				color: green;
			}

De forma similar para aplicar un estilo al último item de la lista ordenada utilizamos la pseudoclase last-child:

			ol li : last-child {
				color: red;
			}

De forma similar para aplicar un estilo al último item de la lista ordenada utilizamos la pseudoclase last-child:

			ol li : last-child {
				color: red;
			}

Pseudoclases aplicadas a selectores

						ol > li:first-child{
							color: #EA2919 ;
						}
	
						ol > li:last-child{
							color: #14D407;
						}

El primer selector utiliza la pseudoclase first-child para dar un color rojo al primer hijo de la lista no ordenada.

El segundo selector mediante la pseudoclase last-child aplica el color verde al texto al ultimo elemento hijo de la lista no ordenada

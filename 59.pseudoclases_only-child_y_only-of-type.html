Pseudoclases: only-child & only-of-type

La pseudoclase only-child se aplica cuando en dicho nivel hay un solo elemento, representa un elemento sin hermanos, esto es lo mismo que utilizar las pseudoclases first-child, last-child o nth-child ( 1 ), nth-last-child ( 1 ), pero con una especificidad menor.

Su sintaxis:

			selector : only-child {
				color : lime;
			}

Por ejemplo: Se cuenta con una lista no ordenada con un unico list item ( li ): 

			<ul id = "fechainicio">
				<li> 10/10/2015 </li>
			</ul>

Luego si especificamos la siguiente regla:

			li : only-child {
				color : blue;
			}

El elemento list item ( li ) se pintará de azul, ya que hay un solo elemento en ese nivel, es decir no cuenta con más hermanos (elementos li que están al mismo nivel jerarquico). Por lo cual el estilo se aplica sin ningun problema.

Si tenemos por ejemplo se cuenta con un contenedor tipo ( div ), el cual tiene definida una propiedad id llamara equipo 1, con 2 hijos, el primero un titulo tipo h1 y el segundo un párrafo:

			<div id = "equipo1">
				<h1> Participantes </h1>
				<p> Pablo Martinez </p>  
			</div>

Y se intenta aplicar la pseudoclase only-child al elemento parrafo ( p ), definiendo un color de texto rojo para este elemento, el resultado será?:

			#equipo1 p:only-child {
				color:red;
			}

Pues el resultado es que no se aplicará el color de texto rojo al párrafo, ya que no es el único elemento en dicho nivel, cuenta con un hermano tipo h1. Por lo tanto no es un unico hijo y por ello la pseudoclase only-child no funciona.

Ahora la pseudoclase only-of-type representa un elemento que no tiene hermanos del mismo tipo. Se aplica cuando en dicho nivel hay un solo elemento de ese tipo, no importa cuantos otros elementos hay de otro tipo en el mismo nivel

Retomando el ejemplo anterior:

			<div id = "equipo1">
				<h1> Participantes </h1>
				<p> Pablo Martinez </p>  
			</div>

Utilizando la misma regla css definida, con la unica diferencia que está vez se utilizará la pseudoclase only-of-type, cual será el resultado?:

			#equipo1 p : only-of-type{
				color : red;
			}

Pues el resultado es que ahora si se aplica el color rojo el párrafo, ya que en ese nivel hay solo un elemento de tipo párrafo, sin importar que tenga hermanos de otro tipo, si la regla es para un elemento que es unico a ese nivel jerárquico, el estilo se aplicará.

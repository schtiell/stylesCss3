Pseudoclases: only-child & only-of-type

Desarrollar una página que muestre una tabla HTML con una fila. Fijar el color amarillo a la fila si tiene solo una fila la tabla. Modificar la tabla HTML agregando más filas y comprobar si ahora no se pinta la primer fila.

Ejemplo:

Tabla de usuarios

id nombre municipio email
01 Joaquin Xalapa isc.jsayago@hotmail.com

Pseudoclases css aplicadas

						tr : only-child {
							background : purple;
						}

Tabla de usuarios

id nombre municipio email
1 joaquin xalapa isc.jsayago@hotmail.com
2 juan tuxpan juan.tuxpan@gmail.com

A esta tabla que está compuesta de 2 filas "table row tr", no se aplica la pseudoclase only-child

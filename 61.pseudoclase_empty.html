Pseudoclase: empty

La pseudoclase empty permite definir una regla css a un elemento html que no tenga elementos hijos y tampoco texto. Su definición en el documento html debe ser completamente vacia

Por ejemplo: 

			<p></p>
			<div></div>

Para probar esta pseudoclase se creará un elemento <div> en forma de rectángulo, con un fondo de color azul, el cual solo será aplicado cuando el div esté vacío. En caso que el div contenga datos no se mostrará el rectángulo amarillo.

Si se agrega algún elemento HTML o texto dentro del <div></div> se observará que no cumple la regla de estilo y se tiene como resultado que no aparece la figura definida de fondo azul.

Importante: Hay que tener cuidado con los elementos HTML que parecen vacíos como por ejemplo:

			<div>
			</div>

Parece vacío el div pero en realidad tiene el salto de línea se está en presencia de texto en el div y por lo tanto no se verifica la regla empty

Para que se aplique la regla de estilo con la pseudoclase empty el div debe estar escrito en el archivo HTML:

			<div></div>

Por tanto se puede decir que la pseudoclase empty representa cualquier elemento que no tenga hijos. Los hijos pueden ser nodos de elemento o texto (incluido el espacio en blanco). Los comentarios o las instrucciones de procesamiento no afectan si un elemento se considera vacío.

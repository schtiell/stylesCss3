CSS: Cascading Style Sheets Pseudoelementos: first-letter & first-line

Al igual que las pseudoclases, los pseudoelementos se añaden a los selectores, pero en cambio, no describen un estado especial sino que, permiten añadir estilos a una parte concreta del documento. Es decir los pseudoelementos a diferencia de las pseudoclases afectan a parte de un elemento HTML y no al elemento completo.

Su sintaxis:

			selector :: pseudoelemento { 
				propiedad: valor; 
			}

Los primeros dos pseudoelementos a analizar en esta explicación son first-letter y first-line.

First-letter

El pseudoelemento first-letter permite seleccionar la primer letra del contenido de un elemento HTML y aplicarle un estilo.

Su sintaxis:

			selector :: first-letter { 
				propiedad: valor; 
			}

La sintaxis para definir un pseudoelemento es distinta a las pseudoclases ya que le antecedemos dos caracteres : : en lugar de uno.

Si no se tuviera los pseudoelementos este problema se debe resolver encerrando el primer caracter de un párrafo con el elemento HTML "span" y asignarle un estilo a dicho elemento. Luego gracias a esto se tiene un contenido del archivo HTML mucho más limpio.

First-line

El pseudoelemento first-line permite seleccionar la primer línea de texto de un elemento HTML para aplicar un estilo. Si tenemos la siguiente página:

El pseudoelemento first-line permite aplicar un estilo a la primer linea del contenido de un elemento HTML. Tengamos en cuenta que la cantidad de datos a mostrar en la primer línea depende del ancho de la ventana del navegador

Tengamos en cuenta que si redimensionamos el navegador solo los caracteres de la primer línea aparecen los estilos aplicados al pseudoelemento

Ejemplo:

Pseudoelemento: first-letter

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Quae exercitationem, illo, culpa hic molestiae rerum possimus unde vitae nam dolorum incidunt excepturi obcaecati. Commodi ad quasi possimus temporibus aliquam molestiae?

Pseudoelemento: first-letter 

							.primeraLetra::first-letter{
								font: oblique bolder 2.5rem 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif;;
								color: lime;
								margin-right: 5px;
							}

Pseudoelemento: first-line

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Nemo quaerat rerum eius rem consequatur quidem veritatis facilis, sit non vitae aliquam debitis sunt, nam deserunt voluptatibus veniam consequuntur illo sint!

Pseudoelemento: first-line 

							.primeraLinea p::first-line{
								font: italic bolder 18px 'Trebuchet MS', 'Lucida Sans Unicode', 'Lucida Grande', 'Lucida Sans', Arial, sans-serif;
								color:red;
							}

Cascading Style Sheets Flexbox: Cajas flexibles

El modulo de flexbox, comunmente llamado Flexbox, permite implementar interfaces de usuario y maquetación de páginas web en forma más sencilla en comparación con la forma en la que se venia haciendo del 2000 al 2014.

Flexbox fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación

Cuando se describe a flexbox como unidimensional se destaca el hecho que flexbox maneja el layout en una sola dimensión a la vez, ya sea como fila o como columna.

La mayoría de los navegadores ya implementan la funcionalidad de flexbox que tambien son conocidas como "Cajas flexibles".

Su gran utilidad y aceptación ha permitido que frameworks como Bootstrap 4 también lo incorporan para su funcionamiento.

Flexbox facilita la maquetación del sitio web y todos sus componentes. También facilita la adaptación de la página a distintos tamaños de pantalla lo que se conoce como: responsive desing

Un flexbox o caja flexible consiste en un elemento contenedor y en su interior una serie de items:

Para definir un elemento como contenedor de cajas flexibles debemos definir la propiedad display con el valor "flex", con esto ya tenemos un caja flexible.

			div {
				display : flex;
			}

La idea principal detrás del diseño flexible es darle al contenedor la capacidad de modificar el ancho/alto (y el orden) de sus elementos para llenar mejor el espacio disponible.

container flex

						.contenedor{
							display: flex;
						}

containers hijos

						.contenedor div{
							margin: 10px;
							padding: 10px;
						}

Luego los tres div contenidos se transforman en cajas flexibles que se redimensionan en ancho y alto automáticamente cuando cambiamos el tamaño del navegador.

Veremos que hay un conjunto de propiedades que nos permiten definir la ubicación, tamaño etc. de los items dentro del contenedor.

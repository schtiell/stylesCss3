CSS: Cascading Style Sheets Flexbox: flex-grow

La propiedad flex-flow, es la propiedad que se aplica a los elementos que estan dentro del contenedor, es decir a los items.

Esta propiedad almacena un valor entero y controla que tanto crecerá el item rellenando el espacio no utilizado por los otros items.

El valor por defecto de la propiedad flex-grow es cero. No tiene unidad de medida ya que indica un factor de crecimiento

Su sintaxis es:

            .item{
                flex-grow: value;
            }

Por lo tanto la propiedad flex-grow define la capacidad de crecimiento de un elemento flexible si es necesario. Acepta valores sin unidades que sirve como proporción. Determina que cantidad de espacio disponible dentro del contenedor flexible debe ocupar el artículo.

Los valores para la propiedad flex-grow no pueden ser negativos.

Ejemplo

flex-direction: row

Caja: 1
Caja: 2
Caja: 3

flex-grow values 

                        .caja2{
                            flex-grow: 5;
                        }
                        
                        .caja3{
                            flex-grow: 2;
                        }

CSS: Cascading Style Sheets Flexbox: flexbox-shrink

Otra propiedad aplicada a los items de una caja flexible se llama flex-shrink y almacena un valor entero no negativo. Este valor representa un factor de encogimiento o de contracci√≥n del item

Cuando no hay mas espacio para repartir en el flexbox container los items comienzan a contraerse para entrar en el contenedor.

Su sintaxis:

            .item {
                flex-shrink: 3;
            }

La propiedad flex-shrink define la propiedad para que un item flexible se encoja si es necesario. Su valor por default es 1, lo que quiere decir que cuando se comienza a reducir el ancho del contenedor los items se encogen en forma proporcional.

Si se desea que uno o mas items no se encojan se le debe asignar a la propiedad flex-shrink el valor 0

Los numeros negatinos no son validos.

Ejemplo

flex-shrink

Caja: 1
Caja: 2
Caja: 3

                        }

                            flex-shrink: 1;

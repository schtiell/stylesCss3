CSS: Cascading Style Sheets Flexbox: flex-basis

Otra propiedad aplicada a los items de una caja flexible es la propiedad flex-basis define el tamaño predeterminado de un elemento antes de que se distribuyan en el espacio restante. La propiedad flex-basis define el ancho con respecto al eje principal si el flex-direction está definido como row, si el flow direction está definido como column el flex-basis representa el height.

El tamaño base que se define en la propiedad flex-basis puede no llegar a cumplirse dependiendo de las propiedades vistas que se aplican a los items. Esta propiedad es fundamental para definir el tamaño de los elementos, pueden emplearse diferentes tipos de medidas absolutas o relativas.

Su sintaxis:

            .item{
                flex-basis: auto /*valor por defecto*/
            }

La propiedad flex-basis almacena el valor por defecto auto, es decir calcula su medida de manera automática, lo que es lo mismo que el item se autodimensiona en función del contenido del elemento.

Si el valor se establece en 0, el espacio adicional alrededor del contenido no se tiene encuenta, si se establece en auto el espacio adicional se distribuye en función de su valor flex-grow.

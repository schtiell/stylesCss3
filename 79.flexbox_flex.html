CSS: Cascading Style Sheets Flexbox: flex (grow, shrink & basis)

La propiedad flex, representa el formato resumido para indicar las propiedades flex-grow, flex-shrink y flex-basis. Estas propiedades debe recordarse que aplican unicamente a los elementos o items de una caja flexible

Su sintaxis debe ser en el siguiente orden:

            .item{
                flex: flex-grow, flex-shrink, flex-basis;
            }

La sintaxis anterior muestra como deben especificarse los 3 valores para un item, aunque se debe recordar que estos tienen valores por defecto, lo que significa que puede inicializarse la propiedad flex con menos de 3 valores. Por ejemplo:

La propiedad flex-basis queda con su valor por defecto auto:

            .item{
                flex: 1 2;
            }

La propiedad flex-shrink queda con su valor por defecto 1. La propiedad flex-grow recibe el valor 1 y flex-basis el valor de 20%

            .item{
                flex: 1 20%;
            }

Los valores se almacenan como 0 0 auto:

            .item{
                flex: none;
            }

La propiedad flex toma todos los valores por default de cada propiedad, es decir (0 1 auto)

            .item{
                flex: auto;
            }

CSS: Cascading Style Sheets Flexbox: align-self

La propiedad align-self permite especificar la alienación de los items dentro del contenedor flexible con respecto al eje transversal.

Asimismo propiedad align-self permite anular la alienación predeterminada o la especificada por la propiedad align-items para los elementos dentro de un contenedor flexible.

Los valores que puede recibir la propiedad align-self son:

  • auto
  • flex-start
  • flex-end
  • center
  • stretch
  • baseline

El valor por defecto de la propiedad align-self es auto

Ejemplo:

flex-direction row

Caja: 1
Caja: 2
Caja: 3
Caja: 4
Caja: 5
Caja: 6

align-self items 

                        .flex-container{
                            display: flex;
                            flex-direction: row;
                            justify-content: space-between;
                            align-items: flex-start;
                        }

                        .caja1{
                            align-self: auto;
                        }
                        
                        .caja2{
                            align-self: baseline;
                        }
                        
                        .caja3{
                            align-self: flex-end;
                        }
                        
                        .caja4{
                            align-self: stretch;
                        }
                        
                        .caja5{
                            align-self: center;
                        }
                        
                        .caja6{
                            align-self: flex-start;
                        }
